## Contributing

Thanks for your interest in contributing!

If you found a bug or want to improve something,
please open a Pull Request directly.

In the PR description:

- describe the problem
- describe the solution

Small and focused PRs are appreciated.

External contributions should target the `contribution` branch.
Changes are promoted to `main` after the trusted validation flow passes.

Because promotions to `main` are typically squash-merged, the shared `contribution` branch may be reset to match `main` afterward. This is expected and only affects the shared staging branch, not your own branch or fork.

Issues are optional and mostly useful for larger discussions.

Thank you!
